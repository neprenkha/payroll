# Workbook Documentation Index

## Overview

This workbook appears to provide a payroll/time-attendance calculator with company/staff settings, statutory contribution tables (KWSP/EPF, SOCSO/Perkeso, EIS), and a holiday list. Monthly sheets are named by month number (e.g., 04, 05) and contain daily time/attendance inputs and calculations.

**Core sheets (expected):** 01, 02, 03, 04, 05, 06, 07, 08, 09, 10, 11, 12, companysettings, staffsettings, kwsp, perkeso, holidays
**Unknown/Legacy sheets:** shifts, settingsarchive, rosterpatterns, rosteroverrides
